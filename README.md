# Simulador de Interés Compuesto (CLI) 📈

Este programa en **Python** ayuda a proyectar el crecimiento de una inversión a través del tiempo. 
Es ideal para visualizar el impacto de las aportaciones mensuales y la tasa de interés.

## 🛠️ ¿Cómo funciona?
El script solicita al usuario:
1. **Capital inicial:** El dinero con el que empiezas.
2. **Aportación mensual:** Cuánto dinero añadirás cada mes.
3. **Tasa anual:** El porcentaje de rendimiento (ej. 11% de CETES).
4. **Tiempo:** Cuántos años quieres proyectar.

## 🚀 Características técnicas
* **Lógica de Interés Compuesto:** Realiza el cálculo mensual dinámico.
* **Manejo de Errores:** Valida que el usuario ingrese números y no texto.
* **Exportación:** Genera un archivo `proyeccion.csv` compatible con Excel.
